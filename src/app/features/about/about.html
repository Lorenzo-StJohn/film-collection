About

Film Collection is a tiny educational project where you can browse movies, search for titles, and manage your own favorites. It’s built entirely with Angular as a hands-on way to practice modern front‑end development.

This app was created as part of the Rolling Scopes School Angular 2026 Q2 course. It’s my playground for experimenting with components, routing, directives, pipes, and signals — all in a real, deployable project.

Why this page exists: I’m actively looking for a mentor in the RS School Angular course to review my code, share feedback, and help me grow as a developer. If you’re a mentor and you’ve found this link — thank you for stopping by! I’d be grateful for any guidance you can offer.

Feel free to explore the app, and check out the source code on GitHub. I’m excited to learn and improve — one commit at a time.
